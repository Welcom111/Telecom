# РТС Телеком
## ИС для анализа финансовых результатов и прогноза
### Практическое задание 4 курс
ИС для анализа финансовых результатов и прогноза
Ключевые возможности
- Загрузка данных из CSV (адаптеры под БД — заготовки).
- KPI: LTM выручка, YoY, валовая/EBIT/чистая маржинальность, CAGR.
- Прогноз выручки, ОПЕКС и чистой прибыли (ETS, месячная сезонность).
- Сценарный анализ (best/base/worst): корректировки роста и инфляции затрат.
- Экспорт прогнозов в CSV.
- Docker-образ и GitHub Actions (pytest).
